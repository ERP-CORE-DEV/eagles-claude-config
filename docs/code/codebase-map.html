Codebase Map

EAGLES Framework file structure with progressive depth levels

1. TL;DR

The EAGLES framework lives in two locations: ~/.claude/ (active runtime with agents, skills, hooks, rules, and MCP config) and C:\.claude\ (reference library with 103+ YAML skills, DSL engine, architecture configs, and HTML docs).

2. Plain English

Think of EAGLES like a well-organized toolbox. The reference library (C:\.claude\) is the master catalog — it has every tool blueprint (skills), recipe book (YAML configs), and a smart machine (DSL engine) that reads shorthand codes. The active runtime (~/.claude/) is your personal workbench — custom tools (agents), house rules (rules/), safety guards (hooks), and phone numbers (MCP servers) for all specialized assistants.
