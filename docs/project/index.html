What is EAGLES?

EAGLES (Enhanced AI-Guided Lean Engineering System) is a universal Claude Code framework that provides:

Quick Start Preview

BASH 
# 1. Clone the EAGLES config
git clone https://github.com/ERP-CORE-DEV/eagles-claude-config
cd eagles-claude-config

# 2. Run the setup script
.\setup_claude.ps1

# 3. Start using skills in any project
claude> /gitleaks              # Scan for secrets
claude> /tdd MyService         # Start TDD workflow
claude> /scaffold-microservice OrderService

For the full guide, see Quickstart.
