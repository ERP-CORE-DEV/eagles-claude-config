Quickstart

Get EAGLES running in under 5 minutes. By the end, you will have persistent agents, skills, and hooks available in every Claude Code session.

✅

Prerequisites

💻

Claude Code CLI

claude --version

🔀

Git

git --version

🐍

Python 3

python3 --version

🟢

Node.js 18+

node --version

1Clone the framework

BASH 
git clone https://github.com/ERP-CORE-DEV/eagles-claude-config.git
cd eagles-claude-config

2Run the setup script

POWERSHELL 
.\setup_claude.ps1

This copies the following to your ~/.claude/ directory:

WhatWhereCount
Agents~/.claude/agents/24
Skills~/.claude/skills/72
Rules~/.claude/rules/10
Hooks~/.claude/hooks/ + settings.json7+1
MCP config~/.claude/.mcp.json10
Settings~/.claude/settings.json1

3Verify the installation

BASH 
claude

You should see the SessionStart hook fire. Run a quick skill:

CLAUDE 
/repo-map

If you see file listings with importance rankings, EAGLES is working.

4Try core skills

TDD workflow

CLAUDE
/tdd implement a CalculateDiscount method that applies 10% for orders over 100 euros

Activates the tdd-guide agent: RED -> GREEN -> REFACTOR

Code review

CLAUDE
/code-review src/backend/Services/MatchingService.cs

Scans for quality, security, and maintainability issues

Secret scanning

CLAUDE
/gitleaks

Runs gitleaks patterns against your repository

Generate docs

CLAUDE
/update-docs official --preview

Dispatches the 5-agent tech writer team

5Understand what happens automatically

Once installed, these hooks fire without any action from you:

WhenWhat happensWhy
You write a .md filedoc-blocker checks if the path is allowedPrevents documentation sprawl
You run git commitgitleaks blocks without prior scanSecrets never reach the repository
You run dotnet test without coveragecoverage-gate prints a reminderMaintains test coverage visibility
You edit a .ts fileTSC check suggests compilationCatches type errors early
Every 25 tool callscontext-rot counter suggests /compactKeeps the context window healthy
You install a dependencydependency-audit reminds /security-scanCatches vulnerable packages
Session endsStop hook confirms persistenceSession continuity
Session startsSessionStart hook loads summaryContext recovery
